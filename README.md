# MackSun — Solar Data from South America

MackSun is a public portal for solar data access, providing solar time series data, solar flare data and high cadence solar observations for scientific research and analysis.

The platform enables users to explore, visualize and download solar datasets collected from observational instruments.

---

## Data Source and Instruments

The data available in MackSun is collected from solar observational instruments located in South America, including:

- POEMAS (Polarization Emission of Millimeter Activity at the Sun)
- SST (Solar Submillimeter Telescope)

These instruments provide high cadence solar radio data and are used for monitoring solar activity and solar flare events.

---

## Available Data

MackSun provides access to multiple types of solar datasets:

- Solar time series data
- Solar flare data
- Solar radio data
- High temporal resolution observations
- Multi-resolution datasets

---

## Data Formats

Datasets are available in standard scientific formats:

- CSV (structured tabular data)
- FITS (Flexible Image Transport System)

These formats allow integration with scientific tools, data pipelines and analysis environments.

---

## Platform Capabilities

Users can:

- Explore solar data from different instruments
- Visualize solar flare events
- Filter datasets by date and temporal resolution
- Download data for offline analysis
